Pacto de la Alianza

¡Por favor, respete las siguientes convenciones y cúmplalas para un mejor mañana para usted y para toda la sociedad!

  • Cumplir con las leyes y regulaciones locales/nacionales
  • La información protegida por la ley no será rastreada (Por ejemplo: Datos personales de privacidad: número de teléfono móvil, número de identificación, etc.)
  • No se rastrearán los datos que requieren inicio de sesión o estado VIP para acceder.
  • No se rastrearán los datos cuya recopilación esté explícitamente prohibida por el sitio web de destino.
  • No se rastrean los datos comerciales principales del sitio web de destino
  • Baja concurrencia, baja frecuencia, no afecta el funcionamiento normal del sitio web de destino.
